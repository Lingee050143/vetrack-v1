Context Block
▸ Token Structure
Primitive → Semantic → Component (3-layer)
DO NOT use Primitive tokens directly on components
Naming: [Category]/[Role]/[Scale] e.g. color/primary/500
/* AI Token은 일반 status(success/error)와 완전 분리 */
▸ Motion Rules
IF interaction = instant feedback (hover, focus) → motion/fast (100ms, ease-out)
IF interaction = state change (open, close) → motion/base (200ms, ease-in-out)
IF interaction = large transition (page, modal) → motion/slow (300ms, ease-in-out)
DO NOT use transition duration > 300ms for UI components
▸ Accessibility
DO NOT use AI State Tokens as text color — backgrounds only
DO NOT use white text on error/500 — 3.76:1 fails AA
Minimum contrast: body text 4.5:1 | large text (18px+) 3:1

Dark Mode Elevation — Shadow 대신 Border 조합

/* Figma Variables는 Shadow Token 미지원 */
/* Dark Mode에서 Elevation 표현 방법 */
L0 (flat): bg = surface/subtle / border = none
L1 (card): bg = surface/base / border = 1px neutral/200
L2 (floating): bg = surface/elevated / border = 1px neutral/300
L3 (modal): bg = surface/base / overlay + border = 1px neutral/200

Context Block — AI 코드 생성 프롬프트 앞에 붙여넣기
▸ Dark Mode Rules
IF theme change → Figma Variables Mode (not Variant, not separate component)
IF component interaction state → State Variant axis (not Mode)
IF Dark Mode elevation → border + surface/elevated combo (Shadow Token not supported)
▸ Key Token Pairs
surface/base: Light=#FFFFFF / Dark=#1E293B
surface/subtle: Light=#F8FAFC / Dark=#0F172A
neutral/900: Light=#0F172A / Dark=#F1F5F9
